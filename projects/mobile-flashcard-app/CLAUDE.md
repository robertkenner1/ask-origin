# Mobile Flashcard App Prototype

## Project Overview
A premium mobile flashcard application prototype that looks and feels 100x better than Anki. Features a modern iOS-inspired design with smooth animations, gesture support, and responsive layout. Pre-loaded with **500+ German B2 level vocabulary words**.

## Key Features
- **Premium Mobile Design**: iPhone-like interface with authentic mobile UI patterns
- **German B2 Vocabulary**: Pre-loaded with 500+ essential B2 level German words
- **Word Categories**: Nouns, verbs, adjectives, abstract concepts, business terms, and common phrases
- **Add New Words**: Input box with auto-fetch from free dictionary APIs
- **Dictionary Integration**: Automatic definition and example retrieval
- **Gesture Support**: Tap to flip cards, swipe to navigate (left/right)
- **Smooth Animations**: 60fps animations with CSS transforms and transitions
- **Touch Feedback**: Haptic feedback support for mobile devices
- **Keyboard Support**: Full keyboard navigation support
- **Responsive Design**: Works on all screen sizes with mobile-first approach
- **Dark Mode**: Automatic dark mode support based on system preferences
- **Progress Tracking**: Visual progress bar and card counter
- **Modern UX**: Clean, minimalist interface with attention to detail

## Technical Implementation
- **HTML5**: Semantic structure with accessibility features
- **CSS3**: Modern CSS with custom properties, grid, flexbox, and animations
- **Vanilla JavaScript**: Clean, performant ES6+ code with classes
- **Dictionary APIs**: Free Dictionary API with WordsAPI fallback
- **Touch Events**: Full gesture support with touch event handling
- **Performance**: Optimized animations and minimal DOM manipulation
- **LocalStorage**: Card progress and user additions persistence

## User Interactions
1. **Tap card** to flip and reveal answer
2. **Swipe right** or click ✅ for correct answer
3. **Swipe left** or click ❌ for incorrect answer
4. **Add Word button** to open dictionary lookup modal
5. **Type word** and click "Fetch Definition" for auto-lookup
6. **Save Card** to add new words to your deck
7. **Keyboard shortcuts**: Space/Enter to flip, arrows to navigate
8. **Mouse hover** adds subtle parallax effect on desktop

## Dictionary Integration
- **Primary API**: Free Dictionary API (https://api.dictionaryapi.dev)
- **Fallback API**: WordsAPI for additional coverage
- **Auto-fetch**: Automatic definition and example retrieval
- **Error Handling**: Graceful fallback and user feedback
- **No API Keys**: Uses completely free APIs
