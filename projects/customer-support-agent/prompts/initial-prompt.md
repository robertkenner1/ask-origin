# Customer Support Agent - Initial Prompt

## Project Description
A customer support chat agent with a modern web interface. This application allows users to interact with an AI-powered support agent that can answer questions based on a knowledge base.

## Features
- Modern chat interface with message history
- Real-time message streaming
- Support for markdown formatting
- Clean, professional UI design
- Responsive layout

## Technology Stack
- HTML/CSS/JavaScript
- Modern CSS Grid/Flexbox layout
- Fetch API for backend communication
- No external frameworks required
