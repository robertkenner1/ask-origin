### Resources

The design system delivers images and semantic colors as `xcassets` and fonts as `otf`. These `xcassets` and `otfs` are located in the `DesignSystem.bundle` inside the ` DesignSystem.xcframework`. Use Build Phases in Xcode and copy `DesignSystem.bundle` to your app resources with a base script.

Note: to use semantic colors or images in Xcode previews that live in a framework, you have to explicitly specify `Bundle`.

See [Grammarly for macOS](https://gitlab.grammarly.io/mobile/GRKeyboard/-/merge_requests/3352) integration example for inspiration. (Requires VPN and access to `GRKeyboard` repository)

#### Semantic palette

The Semantic palette for macOS is integrated using `xcassets` with the Swift API. To use the Semantic palette, include `DesktopSemanticColorPalette.xcassets` in your Xcode project.

Reference colors using the Swift API:

<CodeBlock
  language={"ts"}
  children={`
func setup() {
  // Using Semantic palette tokens
  let color = Colors.semantic(.color(.border(.base(.default)))) // return reference NSColor in .xcassets or fallback color for unsupported macOS versions
  let color2 = Colors.semantic(.color(.border(.base(.default)))), in: Bundle.current)
}

### Elevation

Elevation is available for macOS and IOS platforms. Use elevation tokens with `NSView` entities on macOS platform, and with `UIView` entities on IOS.

##### Example

<CodeBlock language={"ts"} children={"view.setElevation(.elevation300)"} />

### Iconography

Iconography for macOS is available as `xcassets` file. To use icons, include the specific `xcassets` file in your Xcode project.

Reference images using the Swift API:

<CodeBlock
  language={"ts"}
  children={`

### Space

Space tokens are available to use with `NSStackView`. Use the `.setSpacing` method to add a space value. For support using space tokens with additional views, contact us in the [#gds-support](https://grammarly.slack.com/archives/C03MNHYDT5E) Slack channel.

##### Example

<CodeBlock language={"ts"} children={`$0.setSpacing(.space2)`} />

### Typography

Typography is available with the `NSFont` extension:

- `NSFont.grammarlyFont(ofSize: CGFloat, weight: Weight)`
- `NSFont.grammarlyFont(_: TextStyle)`
- `NSFont.grammarlyItalicFont(ofSize: CGFloat, weight: Weight)`
- `NSFont.grammarlyItalicFont(_: TextStyle)`

Or via `Fonts` type:

- `Fonts.grammarlyFont(size: CGFloat, weight: NSFont.Weight, isItalic: Bool = false)`

For fonts to work correctly in your AppKit application you need to include new key in `Info.plist` file:
