# Component Status

Information about Origin component readiness status regarding Figma, Web, Docs, and Accessibility.

Components are available in both Figma and React. The status of each component is tracked in the table below.

You can also interact with each component in the [Origin Storybook](https://uifoundation.gpages.io/origin/) or the [Component Playground](/components/component-playground).

## Status Legend

- **Available**: Component is ready for use
- **Beta**: Component is in beta
- **In progress**: Component is being developed
- **Not yet available**: Component is not yet available
- **Not applicable**: Component does not apply in this context
