## Usage

Use a suggestion toggle to display multiple transform options on a single suggestion card.

**Do not use** a suggestion toggle to allow a user to make multiple selections (instead, use a [checkbox](/components/checkbox)) or select an option from a list within a form (instead, use a [Radio Group](/components/radio-group)).
