# ColorSchemeProvider

**Full path**: GDS/ColorSchemeProvider

**Import**:
```typescript
import { ColorSchemeProvider } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `scheme` | `ColorSchemeOption` | `"auto"` | Color scheme (aka theme) to use. If not provided, the value is detected based on the user's OS settings ("auto"). |
| `target` | `HTMLElement` | `-` | Custom target to attach `data-gds-theme` attribute. Defaults to a container div if not provided. |
| `children` | `ReactNode` | `-` | Contents that make use of the color scheme. |
