# Rating

**Full path**: GDS/Rating

**Import**:
```typescript
import { Rating } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `-` | Text label displayed above the stars to describe the purpose of the rating. |
| `isError` | `boolean` | `false` | When true, displays an error message below the stars and highlights the rating control. |
| `value` | `RatingValue` | `undefined` | The currently selected rating index (0â€“4), where 0 = 1 star, 4 = 5 stars. |
| `isRequired` | `boolean` | `false` | When true, adds a "(Required)" label indicator and marks the field as required for screen readers. |
| `starLabels` | `StarLabels` | `* [ "Not Valuable", "Slightly Valuable", "Moderately Valuable", "Fairly Valuable", "Very Valuable" ]` | An array of five custom labels for the stars, ordered from lowest to highest value. |
| `onChange` | `(value: RatingValue) => void` | `-` | Callback fired when a user selects or changes a rating. |
