# Menu

**Full path**: GDS/Menu

**Import**:
```typescript
import { Menu } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `-` | The content to render in the Menu |
| `activator` | `"more-horizontal" | "more-vertical" | ReactElement` | `"more-vertical"` | The Button or IconButton that opens the menu when activated. |
| `accessibilityLabel` | `string` | `"More"` | Describes the Icon Button's purpose. |
| `initialOpen` | `boolean` | `false` | Determines whether the Menu should be open initially on mount. Mostly for debug purposes. |
| `onClose` | `() => void` | `-` | Event handler that is called when the Menu is closed. |
| `key` | `string | number | Key` | `-` | Unique identifier for the MenuItem |
| `children` | `ReactNode` | `-` | The content to render in the MenuItem |
| `isDisabled` | `boolean` | `-` | Indicates if the item can be activated |
| `icon` | `IconProps["icon"]` | `-` | Icon to display inside on the left side of the MenuItem |
| `onClick` | `() => void` | `-` | Action for the item when activated |
| `href` | `string` | `-` | If specified, children are rendered as a Link with the provided destination. |
| `target` | `"_blank" | "_self" | "_parent" | "_top"` | `-` | Determines where the Link will be opened. |
| `children` | `ReactNode` | `-` | The items to render in the section. |
| `label` | `string` | `-` | Label for the section. |
| `labelDisplay` | `"visible" | "hidden"` | `"visible"` | Can be used to visually hide the label. |
| `role` | `"separator" | "none"` | `-` | The role attribute defines the semantic meaning of the menu separator. |
