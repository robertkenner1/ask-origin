# Badge

**Full path**: GDS/Badge

**Import**:
```typescript
import { Badge } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `getTooltipText` | `(count: number, specialCharBefore?: string, specialCharAfter?: string) => string` | `-` | Format aria label on the Badge component tooltip |
| `tabIndex` | `number` | `0` | Tab index for the Badge component. |
| `tooltipProps` | `Partial<TooltipProps>` | `-` | Props to pass to the Tooltip component. |
| `tooltipContentProps` | `Partial<TooltipContentProps>` | `-` | Props to pass to the TooltipContent component. shadowDom and iFrame environments may needed to override this prop and provide their own root |
