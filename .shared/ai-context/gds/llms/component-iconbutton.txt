# IconButton

**Full path**: GDS/IconButton

**Import**:
```typescript
import { IconButton } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `IconProps["icon"]` | `-` | Icon to display inside the IconButton |
| `accessibilityLabel` | `string` | `-` | Describes the Icon Button's purpose. Also populates the Tooltip, unless otherwise changed through `tooltipContentProps`. |
| `size` | `"small" | "medium" | "large" | "xlarge"` | `'medium'` | Size (height and width) of the Icon Button |
| `tooltipProps` | `Partial<TooltipProps>` | `-` | Props to pass to the Tooltip component. E.g. initiallyOpen, placement, etc |
| `tooltipContentProps` | `Partial<TooltipContentProps>` | `-` | Props to pass to the TooltipContent component. shadowDom and iFrame environments may need to override this prop and provide their own root |
| `variant` | `Exclude<BaseButtonProps["variant"], "critical" | "pro" | "enterprise">` | `'primary'` | Look and feel of the Icon Button |
