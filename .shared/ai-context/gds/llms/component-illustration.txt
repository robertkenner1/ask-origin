# Illustration

**Full path**: GDS/Illustration

**Import**:
```typescript
import { Illustration } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `(typeof illustrationTypes)[number]` | `-` | Which illustration to display. |
| `size` | `"small" | "medium" | "large" | "xlarge"` | `'medium'` | Defines the size of the Illustration. Small (32x32px), Medium (56x56px), Large (80x80px) and xLarge (96x96px) |
| `accessibilityLabel` | `string` | `-` | Used to provide a label for screen reader users. |
| `colorMode` | `"auto" | "light" | "dark"` | `-` | Whether to follow the inherited color scheme (light or dark). |
