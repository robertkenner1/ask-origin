# Icon

**Full path**: GDS/Icon

**Import**:
```typescript
import { Icon } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `accessibilityLabel` | `string` | `-` | Used to provide a label for screen reader users. |
| `variant` | `(typeof variants)[number]` | `'default'` | Styles the Icon according to its use. |
| `isDecorative` | `boolean` | `true` | Sets the value of the `aria-hidden` attribute on the icon. If a non-empty |
| `size` | `"small" | "medium" | "large" | "xlarge" | "xxlarge"` | `'medium'` | Defines the size of the Icon (only intended for Interface icons). |
| `icon` | `{ __brand: "color" | "nocolor"` | `-` | Defines what icon is displayed. Must be an icon function generated by SVGR |
| `resourcePrefix` | `string` | `-` | Provides a prefix for all `id` attributes and ID-based references inside the SVG, |
