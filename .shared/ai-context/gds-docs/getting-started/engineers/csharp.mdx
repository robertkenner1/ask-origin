## Usage

#### Import library

Import `ResourceDictionary` where the library will be used.<CodeBlock
language={'xml'}
children={`<ResourceDictionary xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:gds="clr-namespace:Grammarly.DesignSystem.Core;assembly=Grammarly.DesignSystem">`}
/>

### Border Radius

<BorderRadiusUsage
  language={"xml"}
  usageCode={'CornerRadius="{Binding Source={x:Static gds:BorderRadius.RadiusHalf}}"'}
/>

### Color

<CorePaletteUsage code={`Foreground="{x:Static gds:CoreBrushes.Blue50}"`} />

#### Semantic palette

Semantic palette colors are provided as raw values for the Windows Presentation Foundation (WPF) platform. Switching to other modes requires manually including other palettes.

###### Example

<CodeBlock
  language="xml"
  children={`<SolidColorBrush Color="{x:Static gds:SemanticColors.BaseSurfacePrimaryDefault}" />`}
/>

### Elevation

Elevation for Windows is available as an `xaml` file with ResourceDictionary. To use elevations, include the specific `xaml` file in your project.

#### Example

<CodeBlock
  language="xml"
  children={`
  <ResourceDictionary>
    <ResourceDictionary.MergedDictionaries>
      <ResourceDictionary Source="pack://application:,,,/Grammarly.DesignSystem;component/styles/Elevation.xaml" />
    </ResourceDictionary.MergedDictionaries>
  </ResourceDictionary>`.trim()}
/>

Use elevations to add styles for `Border`:

<CodeBlock language="xml" children={`<Border Style="{ StaticResource Elevation300 }"></Border>`} />

### Iconography

Iconography for Windows is available as an `xaml` file with ResourceDictionary. To use icons, include the **specific** `xaml` file in your project.

##### Example

Reference `xaml` file with images in `Resources.xaml`:

| Resource                         | IconGroup |
| -------------------------------- | --------- |
| DesignSystemInterfaceImages.xaml | Interface |
| DesignSystemFlagImages           | Flag      |
| DesignSystemGbuttonImages        | GButton   |
| DesignSystemGGOImages            | GGO       |
| DesignSystemOutcomeImages        | Outcome   |

<CodeBlock
  language={"xml"}
  children={`
<ResourceDictionary.MergedDictionaries>
    <!--- ... --->

    <utils:SharedResourceDictionary Source="pack://application:,,,/Grammarly.DesignSystem;component/iconography/DesignSystemInterfaceImages.xaml" />
    <!--- ... --->

</ResourceDictionary.MergedDictionaries>
  `.trim()}
/>

Reference images using the C# API:

<CodeBlock
  language={"xml"}
  children={`
<Image Source="{StaticResource gds-icon-interface-apps}" />
  `.trim()}
/>

<CodeBlock
  language="csharp"
  children={`
Application.Current.FindResource("gds-icon-interface-apps");
`.trim()}
/>

### Typography

Typography for Windows is provided as `Grammarly.DesignSystem.Core.Typography` namespace and `.ttf` font files. To use typography, include `ttf` fonts in your project (follow the example below).

##### Example

Include `ttf` fonts:

<CodeBlock
  language={"xml"}
  children={`<FontFamily x:Key="InterFontFamily">pack://application:,,,/Grammarly.DesignSystem;component/typography/#Inter</FontFamily>`}
/>

Use typography tokens in `cs` files:

<CodeBlock
  language={"csharp"}
  children={`using Grammarly.DesignSystem.Core;\n\nTypography.HeadingLarge.FontSize;\nTypography.HeadingLarge.FontWeight;\nTypography.HeadingLarge.LineHeight;\nTypography.HeadingLarge.TokenName;`}
/>

Use typography tokens in `xaml` files:

<CodeBlock
  language={"ts"}
  children={`// xmlns:gds="clr-namespace:Grammarly.DesignSystem.Core;assembly=Grammarly.DesignSystem"\n\nFontSize="{x:Static gds:Typography.HeadingLarge.FontSize}" FontWeight="{x:Static gds:Typography.HeadingLarge.FontWeight}"`}
/>
