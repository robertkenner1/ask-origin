# Extension

How the Grammarly widget works in web browsers

:::caution Editor’s note: 
Your Grammarly widget will differ from the end user’s experience due to internal experiments. GDS documents address the current user-facing UI.
:::

## Position

When Grammarly is running and checking text, the widget is fixed in the lower-right corner of most websites’ writing surfaces.

<div class="diagram p-8">
  <img src={useBaseUrl("/img/patterns/widget/widget-extension-position-textbox.svg")} alt="Diagram showing the Grammarly widget positioned in the lower right corner of a textbox." />
</div>

<br />

In Google Docs, the widget is fixed in the lower-right corner of the full browser window. Though it won’t get in the way of the writing surface, it may overlap some other Google Docs elements, like comments. 

<div className="diagram p-8">
  <img
    src={useBaseUrl("/img/patterns/widget/widget-extension-position-gdocs.svg")}
    alt="Diagram showing the Grammarly widget positioned in the lower right corner of a browser window."
  />
</div>

## States

### Active

When no suggestions are available, the widget appears in the active state. A user can activate generative AI features with the lightbulb Icon Button or more suggestions with the G logomark Icon Button.

<div className="diagram p-8">
  <img
    src={useBaseUrl("/img/patterns/widget/widget-extension-active.svg")}
    alt="The Grammarly widget displays the Generative AI icon and Grammarly logomark icon in green when active."
  />
</div>

### Expanded

When a user hovers on or moves focus to the widget, additional actions can appear. By default, an Icon Button appears on the left that allows a user to turn off Grammarly on a specific page. 

When focus moves away from the widget, it returns to its previous state. 

<div className="diagram p-8">
  <img
    src={useBaseUrl("/img/patterns/widget/widget-extension-expanded.svg")}
    alt="The expanded state of the Grammarly widget showing the logout button."
  />
</div>

### Loading

When a user is typing or editing, a loading spinner appears inside the G logomark Icon Button. The spinner remains until the total number of each suggestion type is determined. Only then will a suggestion count appear.

<div className="diagram p-8">
  <div className="p-8">
    <img
      src={useBaseUrl("/img/patterns/widget/widget-extension-loading-critical.svg")}
      alt="The Grammarly widget loading state animating over a red critical suggestion background."
    />
  </div>
  <div className="p-8">
    <img
      src={useBaseUrl("/img/patterns/widget/widget-extension-loading-active.svg")}
      alt="The Grammarly widget loading state animating over a green active background."
    />
  </div>
</div>
<p className="caption">
  The background color of the loading state changes based on whether critical suggestions are available when a user starts typing. If critical suggestions are available, the background color is red. If not, the background color is green.
</p>

### Tone detection

If generative AI features are turned off, an emoji can appear next to the G logomark Icon Button. The emoji communicates a tone detected in the user’s writing. Activating the emoji Icon Button displays more information and any additional tones that were detected.

<div className="diagram p-8">
  <img
    src={useBaseUrl("/img/patterns/widget/widget-extension-tone.svg")}
    alt="An smiling emoji with sunglasses replaces the G logomark Icon Button."
  />
</div>

### Inactive

When the widget is inactive, the G logomark appears with a subdued background. This state is usually activated when documents exceed a certain length or when a user is not the document's owner.

    alt="A gray G logomark Icon Button indicates that the Grammarly widget is inactive."
  />
</div>
