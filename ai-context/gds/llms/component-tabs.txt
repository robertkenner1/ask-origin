# Tabs

**Full path**: GDS/Tabs

**Import**:
```typescript
import { Tabs } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `-` | Passes the Tab List and Tab Panels. |
| `defaultSelectedTab` | `string` | `-` | The first Tab is activated by default unless using this property to override the value. This corresponds with the Tab `id` prop. We do not recommend overriding this behavior. |
| `onChange` | `(id: string) => void` | `-` | Event handler that is called when a Tab is selected. |
| `size` | `(typeof sizes)[number]` | `"medium"` | Determines the size of the Tabs |
| `accessibilityLabel` | `string` | `-` | Used to describe the theme of the entire set of Tabs. |
| `children` | `ReactNode` | `-` | Used for the Tab subcomponents. Must include at least two. |
| `bottomBorder` | `"default" | "none"` | `"default"` | Determines if there is a border below the Tabs that stretches the full width of the container. |
| `width` | `"full" | "fixed"` | `'fixed'` | Width of the TabList |
| `id` | `string` | `-` | A unique ID that must match to the corresponding `TabPanel`. |
| `label` | `string` | `-` | Used for the text displayed in an individual Tab. |
| `children` | `ReactNode` | `-` | Used to implement custom content for inside the Tab. |
| `children` | `ReactNode` | `-` | Used for the main content of the `TabPanel`. |
| `id` | `string` | `-` | A unique ID that must match to the corresponding `Tab`. |
