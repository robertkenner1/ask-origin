# Button

**Full path**: GDS/Button

**Import**:
```typescript
import { Button } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `-` | Content for inside the Button |
| `text` | `string` | `-` | Text inside the Button |
| `iconStart` | `IconProps["icon"]` | `-` | Icon to display inside on the left of the Button. For Premium Buttons, only the Premium Icon is allowed. |
| `iconEnd` | `IconProps["icon"]` | `-` | Icon to display inside on the right of the Button. For Premium Buttons, only the Premium Icon is allowed. |
| `isLoading` | `boolean` | `false` | Whether to show a circular loader, such as when the associated Form has been submitted. |
| `form` | `string` | `-` | Link form submit button to a form in case the button is used outside of the form |
