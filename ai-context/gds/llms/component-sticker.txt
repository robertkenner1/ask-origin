# Sticker

**Full path**: GDS/Sticker

**Import**:
```typescript
import { Sticker } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `discount` | `number` | `-` | Percent discount for the sticker for Grammarly Pro. |
