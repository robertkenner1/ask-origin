# ScreenReaderOnly

**Full path**: GDS/ScreenReaderOnly

**Import**:
```typescript
import { ScreenReaderOnly } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `-` | The content to be made visible only to screen-readers. |
| `as` | `AsOptions` | `'span'` | The rendered HTML element. |

## Examples

### Default

```tsx
(
    <Text as="span">
      <ScreenReaderOnly>Error: </ScreenReaderOnly> File not found
    </Text>
  )
```
