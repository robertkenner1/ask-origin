# SearchField

**Full path**: GDS/SearchField

**Import**:
```typescript
import { SearchField } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `-` | Displays the text label for the Search Field. |
| `labelDisplay` | `"visible" | "hidden"` | `-` | Can be used to visually hide the label when another element on the page is acting as the visual label for the TextField. |
| `name` | `string` | `-` | Unique name for this input, used for form validation. |
| `placeholder` | `string` | `-` | Adds placeholder text to the input, if needed. Recommended not to use. |
| `value` | `string` | `-` | To implement as controlled component, the `value` prop can be used to maintain the inputâ€™s value |
| `isDisabled` | `boolean` | `false` | When true, indicates that the Search Field cannot be focused, changed, or interacted with. |
| `accessibilityLabelClearButton` | `string` | `-` | Can be used to override the aria-label on the clear icon button |
| `size` | `"medium" | "large" | "xlarge"` | `"medium"` | Controls the height of the Search Field. |
| `onSubmit` | `(value: string) => void` | `-` | Callback fired when the Enter key is pressed and focus is within the Search Field input |
| `onClear` | `() => void` | `-` | Activated when the Clear iconButton at the end of the Search Field is pressed. |
