# Tooltip

**Full path**: GDS/Tooltip

**Import**:
```typescript
import { Tooltip } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | If you plan to manually control the Tooltip state, use `open` and `onOpenChange`. |
| `onOpenChange` | `Dispatch<SetStateAction<boolean>>` | `-` | Event handler that is called when the open state is internally changed. Use this to manually control the `open`. |
| `onShow` | `() => void` | `-` | - |
| `onHide` | `() => void` | `-` | - |
