# Popover

**Full path**: GDS/Popover

**Import**:
```typescript
import { Popover } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `dismissOnOutsideClick` | `boolean` | `true` | Determines whether the user can click outside the Popover to close it. |
| `initialOpen` | `boolean` | `false` | Determines whether the Popover should be open initially when its Popover.Anchor is mounted. |
| `placement` | `Placement` | `'top'` | Indicates placement of the Popover. If not specified, the Popover will be placed automatically based on position priority. |
| `open` | `boolean` | `false` | If you plan to manually control the Popover state, use `open` with `onOpenChange` options. |
| `onOpenChange` | `Dispatch<SetStateAction<boolean>>` | `-` | Event handler that is called when the open state is changed. Use this to manually control `open`. |
| `children` | `ReactNode` | `-` | `PopoverAnchor` controls where the Popover appears. It can also activate the Popover. |
| `onShow` | `() => void` | `-` | - |
| `onHide` | `() => void` | `-` | - |
| `children` | `ReactNode` | `-` | The UI element that the Popover anchors itself to. If this also activates the Popover, it must be a Button or Icon Button. |
| `accessibilityLabel` | `string` | `-` | Used to give context to people using screen readers about the purpose of this Popover. |
| `children` | `ReactNode` | `-` | Contents of the popover. |
| `title` | `string` | `-` | A visible title displayed in bold inside the Popover. |
| `root` | `HTMLElement | null | undefined` | `-` | You can explicitly set the root element for the portal. |
| `illustration` | `ReactNode` | `-` | A decorative image or illustration that appears above the title in the Popover. |
| `width` | `number | string` | `-` | Sets a fixed width for the Popover. Can be a number for pixels or a string for percentages, like “75%”. Range is 225 to 480 px. |
