# Toast

**Full path**: GDS/Toast

**Import**:
```typescript
import { Toast } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | `-` | The text shown in the Toast. |
| `variant` | `ToastVariants` | `"default"` | Changes the icon to convey a system status. |
| `primaryAction` | `ReactNode` | `-` | A Button representing the primary action to take on the Toast. Only use Ghost Buttons inside Toasts. |
| `secondaryAction` | `ReactNode` | `-` | A Button representing the secondary action to take on the Toast. Only use Ghost Buttons inside Toasts. |
| `onClose` | `() => void` | `-` | Event handler that is called when the Close button is clicked. |
