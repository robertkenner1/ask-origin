# PortalContainerProvider

**Full path**: GDS/PortalContainerProvider

**Import**:
```typescript
import { PortalContainerProvider } from '@grammarly/design-system';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `default` | `HTMLElement` | `-` | The default outlet where portals should be rendered. |
| `modal` | `HTMLElement` | `-` | The outlet where Modal portals should be rendered. Leave empty to just use `default`. |
| `tooltip` | `HTMLElement` | `-` | The outlet where Tooltip portals should be rendered. Leave empty to just use `default`. |
| `popover` | `HTMLElement` | `-` | The outlet where Popover portals should be rendered. Leave empty to just use `default`. |
